# Express shop

A CRUD app where user can list their products and buy products from other users.

## Technologies and features

- Nodejs
- Express
- Ejs
- MongoDB
- Mongoose ODM
- Session & Cookie
- Authentication
- User input validation with Zod.
- Image file management with Multer.
